Homework Exercises

Exercise 1: Grading System with Validation

What to do:

  • Create a grading system similar to the one in hands-on
  • Use if/else if/else structure to assign grades
  • Test with marks = 85 (should show "Grade: B")
let marks = 85;
if (marks >= 90) {
  console.log("Grade: A");
} else if (marks >= 80) {
  console.log("Grade: B");
} else if (marks >= 70) {
  console.log("Grade: C");
} else {
  console.log("Grade: F");
}

// Test with different values:
// marks = 95 → Grade: A
// marks = 85 → Grade: B
// marks = 72 → Grade: C
// marks = 50 → Grade: F

Exercise 2: Day Type with Switch Statement

What to do:

  • Use a switch statement to identify weekdays and weekends
  • Use intentional fall-through (don't break between related cases)
  • Test with: "Wednesday" (weekday), "Saturday" (weekend)
let day = "Wednesday";
switch(day) {
  case "Monday":
  case "Tuesday":
  case "Wednesday":
  case "Thursday":
  case "Friday":
    console.log("Weekday");
    break;
  case "Saturday":
  case "Sunday":
    console.log("Weekend");
    break;
  default:
    console.log("Invalid day");
}

// Test cases:
// "Monday" → Weekday
// "Wednesday" → Weekday
// "Saturday" → Weekend
// "Holiday" → Invalid day

Exercise 3: Voting Eligibility with Ternary

What to do:

  • Use a ternary operator to determine voting eligibility
  • Show: "Can vote" if age >= 18, else "Cannot vote"
  • Test with age = 17 (should show "Cannot vote")
let age = 17;
console.log(age >= 18 ? "Can vote" : "Cannot vote");

// Test with different ages:
// age = 17 → Cannot vote
// age = 18 → Can vote
// age = 25 → Can vote
