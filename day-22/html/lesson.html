Day 22 — Map / Set / WeakMap / WeakSet

Lesson

Open DevTools → Console, or run node day-22/js/lesson.js. Source: ../js/lesson.js.

1 Map — better keyed storage

Keys can be any type (including objects). Iteration follows insertion order. Use .size, for...of for entries.

const m = new Map();
m.set("name", "Priya");
m.set(42, "the answer");
m.set(true, "a boolean key");
const userObj = { id: 1 };
m.set(userObj, "metadata for this object");

const m2 = new Map([
  ["a", 1],
  ["b", 2],
]);
for (const [key, value] of m2) {
  console.log(key, value);
}

2 Object ↔ Map conversion

Object.entries / Object.fromEntries bridge plain objects. map.forEach((value, key) => …) — value comes first.

const map = new Map(Object.entries(obj));
const back = Object.fromEntries(map);

for (const key of map.keys()) { ... }
for (const val of map.values()) { ... }
for (const [k, v] of map.entries()) { ... }

map.forEach((value, key) => console.log(key, "=", value));

// Follow-along: two different { id: 1 } literals → m.size === 2
const k1 = { id: 1 };
const k2 = { id: 1 };
m.set(k1, "a");
m.set(k2, "b");

3 Set — unique values

Deduplicates primitives; object identity for objects — two literals are two entries.

const s = new Set();
s.add("a");
s.add("a"); // ignored

const tags = new Set(["js", "react", "js", "node"]);
const uniq = [...new Set([1, 2, 2, 3])];

const set = new Set();
set.add({ id: 1 });
set.add({ id: 1 }); // both kept — different objects

../js/lesson.js — Topic 3

4 Set operations

Union / intersection / difference via spread + filter + has. Modern engines may offer Set.prototype.union etc. (ES2025+).

const union = new Set([...a, ...b]);
const inter = new Set([...a].filter((x) => b.has(x)));
const diff = new Set([...a].filter((x) => !b.has(x)));

../js/lesson.js — Topic 4

5 When to use which

String-keyed config → Object. Ordered list → Array. Frequent add/remove with any key → Map. Membership / dedup → Set. Cache keyed by object → Map (or WeakMap for GC-friendly DOM metadata).

const cache = new Map();
function fetchWithCache(req) {
  if (cache.has(req)) return cache.get(req);
  const result = doFetch(req);
  cache.set(req, result);
  return result;
}

const allTags = new Set();
posts.forEach((p) => p.tags.forEach((t) => allTags.add(t)));

../js/lesson.js — Topic 5

6 WeakMap & WeakSet

Weak references: keys must be objects; not iterable. When the key object is GC'd, the entry disappears — good for DOM node metadata and "seen" object sets without leaks.

const wm = new WeakMap();
wm.set(user, { lastSeen: Date.now() });

const dataForElement = new WeakMap();
function attachData(el, data) {
  dataForElement.set(el, data);
}

const seen = new WeakSet();
function process(item) {
  if (seen.has(item)) return;
  seen.add(item);
}
