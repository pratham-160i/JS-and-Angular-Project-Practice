Day 21 — Immutability & Safe State Updates

Hands-on

Task 1 Prove the Shallow Clone Bug
  • Create a shallow copy using spread syntax.
  • Mutate a nested object.
  • Observe the mutation leak.
  • Fix it using structuredClone(). 
const orig = {

  name: "Priya",

  addr: {

    city: "Jaipur"
  }
};

const copy = {

  ...orig
};

copy.addr.city = "Mumbai";

console.log(
  orig.addr.city
);


// Deep clone version

const deepCopy =
  structuredClone(orig);

deepCopy.addr.city = "Delhi";

console.log(
  orig.addr.city
);

console.log(
  deepCopy.addr.city
);
// Spread only performs
// a shallow clone.
//
// Nested objects still share
// the same reference.
//
// structuredClone()
// creates a deep copy.

Task 2 Immutable Nested Update
  • Update nested state without mutation.
  • Use spread syntax only.
  • Verify the original state remains unchanged. 
const state = {

  user: {

    name: "Priya",

    prefs: {

      theme: "light",

      lang: "en"
    }
  }
};

const newState = {

  ...state,

  user: {

    ...state.user,

    prefs: {

      ...state.user.prefs,

      theme: "dark"
    }
  }
};

console.log(
  state.user.prefs.theme
);

console.log(
  newState.user.prefs.theme
);
// Immutable updates create
// new objects at every changed level.
//
// Original state remains untouched.

Task 3 Update One Item in an Array
  • Update a single task immutably.
  • Use .map() and spread syntax.
  • Ensure original arrays never mutate. 
const tasks = [

  {

    id: 1,

    title: "Learn JS",

    done: false
  },

  {

    id: 2,

    title: "Build app",

    done: false
  }
];

function toggleDone(tasks, id) {

  return tasks.map((task) => {

    if (task.id === id) {

      return {

        ...task,

        done: !task.done
      };
    }

    return task;
  });
}

const updated1 =
  toggleDone(tasks, 1);

const updated2 =
  toggleDone(tasks, 2);

console.log(tasks);

console.log(updated1);

console.log(updated2);
// map() creates a new array.
//
// Spread creates a new object
// only for the updated item.

Try the steps, then compare with ../js/hands-on.js.

Bonus Safe Deep Read
  • Read deeply nested properties safely.
  • Use optional chaining and nullish coalescing.
  • Prevent runtime crashes. 
const data = {

  user: {

    name: "Priya",

    profile: {

      city: null
    }
  }
};

const city =

  data?.user?.profile?.city

  ?? "Unknown";

console.log(city);


const bioLength =

  data?.user?.profile?.bio?.length

  ?? 0;

console.log(bioLength);


// Try with empty object

const emptyData = {};

console.log(

  emptyData?.user?.profile?.city

  ?? "Unknown"
);

console.log(

  emptyData?.user?.profile?.bio?.length

  ?? 0
);
// Optional chaining prevents
// crashes from undefined/null.
//
// Nullish coalescing provides
// safe fallback values.
