Hands-on Tasks

Task 1: Cart Manipulation

What to do:

  • Start with: const cart = ["bread", "milk", "eggs"];
  • Use push() to add "butter". Then use unshift() to add "rice" at the start. Log the cart.
  • Use pop() to remove the last item. Log what was removed and the new cart.
  • Use splice(1, 1) to remove one item starting at index 1. Log the result.
const cart = ["bread", "milk", "eggs"];

cart.push("butter");      // Add butter at end
cart.unshift("rice");     // Add rice at start
console.log(cart);        // ["rice", "bread", "milk", "eggs", "butter"]

const removed = cart.pop();
console.log(removed);     // "butter"
console.log(cart);        // ["rice", "bread", "milk", "eggs"]

cart.splice(1, 1);        // Remove 1 item at index 1 ("bread")
console.log(cart);        // ["rice", "milk", "eggs"]

Task 2: Filter Passing Scores

What to do:

  • Declare: const scores = [88, 42, 75, 60, 91, 39, 55, 70];
  • Use filter() to create a new array of scores that are >= 60 (passing). Log it.
  • Use find() to get the FIRST failing score (< 60). Log it.
  • Use every() to check if ALL scores are passing. Log the result.
  • Bonus: Use some() to check if ANY score is above 90.
const scores = [88, 42, 75, 60, 91, 39, 55, 70];

const passing = scores.filter(score => score >= 60);
console.log(passing);     // [88, 75, 60, 91, 70]

const firstFail = scores.find(score => score < 60);
console.log(firstFail);   // 42

const allPassing = scores.every(score => score >= 60);
console.log(allPassing);  // false

const anyAbove90 = scores.some(score => score > 90);
console.log(anyAbove90);  // true

Task 3: Map Prices with GST

What to do:

  • Declare: const prices = [100, 250, 500, 1200, 80];
  • Use map() to create a new array with 18% GST added to each price
  • Log both the original and new array � the original should be unchanged
  • Bonus: Round each value to 2 decimal places using .toFixed(2)
const prices = [100, 250, 500, 1200, 80];

const withGST = prices.map(price => price * 1.18);
console.log("Original:", prices);        // [100, 250, 500, 1200, 80]
console.log("With GST:", withGST);       // [118, 295, 590, 1416, 94.4]

const withGSTRounded = withGST.map(price => price.toFixed(2));
console.log(withGSTRounded);             // ["118.00", "295.00", "590.00", "1416.00", "94.40"]

Bonus: Reduce to Total

What to do:

  • Declare: const expenses = [250, 800, 120, 50, 1500, 75];
  • Use reduce() to calculate the total of all expenses. Log it.
  • Use reduce() again to find the highest single expense. Log it.
  • Bonus: Combine filter() + reduce() to total only expenses above 100.
const expenses = [250, 800, 120, 50, 1500, 75];

const totalExpense = expenses.reduce((total, expense) => total + expense, 0);
console.log(totalExpense);  // 2795

const highestExpense = expenses.reduce((max, expense) => expense > max ? expense : max, expenses[0]);
console.log(highestExpense);  // 1500

const totalAbove100 = expenses
  .filter(expense => expense > 100)
  .reduce((total, expense) => total + expense, 0);
console.log(totalAbove100);  // 2670
