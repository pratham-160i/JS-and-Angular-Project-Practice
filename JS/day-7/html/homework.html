Homework Exercises

Exercise 1: Map Greetings

What to do:

  • Given an array of names: ["Priya", "Aarav", "Riya"]
  • Use map() to create a new array of greetings: "Hello, Priya", "Hello, Aarav", etc.
const names = ["Priya", "Aarav", "Riya"];
const greetings = names.map(name => `Hello, ${name}`);
console.log(greetings);
// ["Hello, Priya", "Hello, Aarav", "Hello, Riya"]

Exercise 2: Filter Even and Sum

What to do:

  • Given an array: [1, 2, 3, 4, 5, 6, 7, 8]
  • Filter the array to only EVEN numbers
  • Then use reduce() to sum them
const nums = [1, 2, 3, 4, 5, 6, 7, 8];

const evenSum = nums
  .filter(n => n % 2 === 0)
  .reduce((sum, n) => sum + n, 0);

console.log(evenSum);  // 20 (2+4+6+8)

Exercise 3: Find Maximum (Two Ways)

What to do:

  • Given: [3, 1, 4, 1, 5, 9, 2, 6]
  • Find the max using Math.max(...arr)
  • Find the max using reduce()
const arr = [3, 1, 4, 1, 5, 9, 2, 6];

const max1 = Math.max(...arr);
const max2 = arr.reduce((max, n) => n > max ? n : max, arr[0]);

console.log(max1);  // 9
console.log(max2);  // 9

Exercise 4: Calculate Average

What to do:

  • Write a function that returns the average of an array
  • Use reduce() to calculate the sum, then divide by length
  • Test with: [10, 20, 30] (should return 20)
function average(arr) {
  const sum = arr.reduce((total, n) => total + n, 0);
  return sum / arr.length;
}

console.log(average([10, 20, 30]));  // 20
