Homework Exercises

Exercise 1: Convert Kilometers to Miles

What to do:

  • Write a function kmToMiles(km)
  • Formula: km * 0.621
  • Test with: 10 km
function kmToMiles(km) {
  return km * 0.621;
}

console.log(kmToMiles(10));   // 6.21 miles

Exercise 2: GST Amount Calculator

What to do:

  • Write a function gstAmount(price, rate = 18)
  • Return the GST tax amount (not the total)
  • Use a default GST rate of 18%
  • Test with: price = 1000 (should use default rate)
function gstAmount(price, rate = 18) {
  return (price * rate) / 100;
}

console.log(gstAmount(1000));    // 180 (18% GST)

Exercise 3: Full Name Concatenation

What to do:

  • Write a function fullName(first, last)
  • Return the full name as one string combining first and last names
  • Test with: "Pratham" and "Krishna"
function fullName(first, last) {
  return `${first} ${last}`;
}

console.log(fullName("Pratham", "Krishna"));  // Pratham Krishna

Exercise 4: Check if Adult

What to do:

  • Write a function isAdult(age)
  • Return true if age >= 18, otherwise false
  • Test with: age = 20
function isAdult(age) {
  return age >= 18;
}

console.log(isAdult(20));   // true
console.log(isAdult(15));   // false
console.log(isAdult(18));   // true
