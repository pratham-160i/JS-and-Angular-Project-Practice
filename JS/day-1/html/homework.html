Homework Exercises

Complete these exercises at home. Open the browser console and test your code.

Exercise 1: Create and Run a File

What to do:

  • Create a file day1.js
  • Write 5 different console.log statements in it
  • Run it with node day1.js in your terminal
console.log("Welcome to Day 1");
console.log(42);
console.log("Learning JavaScript is fun!");
console.log(10 + 20);
console.log("The answer is " + 42);

Exercise 2: Declare Variables

What to do:

  • Declare 3 variables — one const for your name, one let for your age, one const for your city
  • Log all three
const myName = "Pratham";
let myAge = 23;
const myCity = "Jaipur";

console.log(myName);
console.log(myAge);
console.log(myCity);

Exercise 3: Try Reassigning Variables

What to do:

  • Try reassigning each of the 3 variables from Exercise 2
  • Note which throws an error and why
const myName = "Pratham";
let myAge = 22;
const myCity = "Jaipur";

// This works - let can be reassigned
myAge = 23;
console.log(myAge);           // 23

// This fails - const cannot be reassigned
// myName = "Rahul";          // Error: Assignment to constant variable

// This fails - const cannot be reassigned
// myCity = "Delhi";          // Error: Assignment to constant variable

console.log("const cannot be reassigned - that's why it's safe for fixed values!");
console.log("let can be reassigned - use it for values that change!");
