Hands-on Tasks

Task 1: Template Literal Sentence Builder

What to do:

  • Declare: let product = "Laptop";, let price = 50000;, let tax = 0.18;
  • Use a template literal to build the sentence: "The Laptop costs ?50000 + ?9000 GST = ?59000"
  • Compute the GST and total INSIDE the template literal � don't pre-compute
  • Bonus: Include a multiline version with each part on its own line
let product = "Laptop";
let price = 50000;
let tax = 0.18;

console.log(`${product} costs ?${price} + ?${price * tax} GST = ?${price + (price * tax)}`);

// Bonus: Multiline template literal
let multiline = `${product} costs 
?${price} + ?${price * tax} GST = 
?${price + (price * tax)}`;

console.log(multiline);

Task 2: Array and Object Destructuring

What to do:

  • Use this array: const scores = [88, 75, 92, 60, 45];
  • Destructure the first two values into top and second. Collect the rest into others
  • Use this object: const user = { name: "Anaya", age: 21, city: "Jaipur" };
  • Destructure name, rename age to userAge, and pull out city
  • Log all extracted variables
let scores = [88, 75, 92, 60, 45];
let [top, second, ...rest] = scores;
console.log("Top: " + top + ", Second: " + second + ", Rest: " + rest);
// Top: 88, Second: 75, Rest: 92,60,45

let user = { name: "Anaya", age: 21, city: "Jaipur" };
let { name, age } = user;
console.log(`${name} is ${age} years old`);
// Anaya is 21 years old

Task 3: Spread to Merge Arrays and Objects

What to do:

  • Use spread operator to merge arrays: [1, 2, 3] with [4, 5]
  • Merge objects with spread: const defaults = { theme: "light", lang: "en" }; and const userPrefs = { theme: "dark" };
  • User preferences should override defaults, but unchanged defaults remain
let arr1 = [1, 2, 3];
let arr2 = [...arr1, 4, 5];
console.log("Merged array: " + arr2);
// Merged array: 1,2,3,4,5

let defaults = { theme: "light", lang: "en" };
let userPrefs = { theme: "dark" };
let merged = { ...defaults, ...userPrefs };
console.log("Merged settings: ", merged);
// { theme: "dark", lang: "en" }
