Control Flow Statements

Control flow determines which code gets executed based on conditions.

if...else Statement

switch Statement

let day = 3;
switch(day) {
  case 1:
    console.log("Monday");
    break;
  case 3:
    console.log("Wednesday");
    break;
  default:
    console.log("Other day");
}
