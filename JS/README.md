# Basic-and-Advance-Javascript

A personal record of progress through a full JavaScript curriculum — from absolute basics to advanced browser concepts — built as interactive, browser-based practice pages you can open and run without any setup.

## Live practice site (GitHub Pages)

After you enable GitHub Pages on this repository, the site will be available at:

**https://pratham-160i.github.io/JS-and-Angular-Project-Practice/JS/**

Use the **Curriculum** panel on the right sidebar to jump between days and pages.

**Local setup:** Clone the repo and open `index.html` directly in your browser — no server needed.

## What you are learning

This curriculum goes from your first line of JavaScript to advanced engine-level behaviour.

## How each day is structured

Every day has three pages, in order:

- **`lesson.html`** — Explanations and demo code you can read and experiment with in DevTools
- **`hands-on.html`** — Short exercises during a study session
- **`homework.html`** — Take-home practice

Scripts for each day live in `day-N/js/`. Shared styles are in `shared/css/style.css`.
