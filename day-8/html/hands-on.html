Task 1: Build a Student Object

What to do:

  • Create an object student with properties: name = "Ananya", age = 21, city = "Jaipur", course = "BTech", marks (array of 3 numbers)
  • Log the student. Then log just the name, age, and the FIRST mark using dot notation
  • Add a new property email = "anaya@example.com". Update age to 22. Delete city
  • Log the student again to confirm changes
const student = {
    name: "Ananya",
    age: 21,
    city: "Jaipur",
    course: "BTech",
    marks: [60, 70, 80]
}

console.log(student);
console.log(student.name);       // Ananya
console.log(student.age);        // 21
console.log(student.marks[0]);   // 60

student.email = "anaya@example.com";
student.age = 22;
delete student.city;

console.log(student);  // city is removed, age is 22, email is added

Task 2: Method with this

What to do:

  • Create an object bankAccount with: holder = "Aarav", balance = 5000
  • Add a method deposit(amount) that adds to balance and returns the new balance
  • Add a method withdraw(amount) that subtracts (only if balance is enough) and returns the new balance, or returns "insufficient balance"
  • Test: deposit 1000, withdraw 2000, withdraw 10000
const bankAccount = {
    holder: "Aarav",
    balance: 5000,

    deposit(amount) {
        this.balance += amount;
        return this.balance;
    },

    withdraw(amount) {
        if (amount <= this.balance) {
            this.balance -= amount;
            return this.balance;
        } else {
            return "insufficient balance";
        }
    }
};

console.log(bankAccount.deposit(1000));     // 6000
console.log(bankAccount.withdraw(2000));    // 4000
console.log(bankAccount.withdraw(10000));   // insufficient balance

Task 3: Destructuring

What to do:

  • Use this object: const product = { id: 101, name: "Laptop", price: 60000, brand: "Dell", stock: 5 };
  • Destructure name and price into variables and log them
  • Destructure brand but rename it to make. Log it.
  • Destructure warranty with a default value of "1 year" (since the property doesn't exist). Log it.
const product = {
  id: 101,
  name: "Laptop",
  price: 60000,
  brand: "Dell",
  stock: 5
};

const { name, price } = product;
console.log(name);       // Laptop
console.log(price);      // 60000

const { brand: make } = product;
console.log(make);       // Dell

const { warranty = "1 year" } = product;
console.log(warranty);   // 1 year (default)

Bonus: Object as Iterable

What to do:

  • Use the student object from Task 1 (after modifications)
  • Use Object.keys() to log all keys
  • Use Object.values() to log all values
  • Use Object.entries() with forEach() to log each property as "key: value"
  • Count how many properties the object has
// After Task 1 modifications, student object has: name, age, course, marks, email

const keys = Object.keys(student);
console.log(keys);  // ["name", "age", "course", "marks", "email"]

const values = Object.values(student);
console.log(values);  // ["Ananya", 22, "BTech", [60, 70, 80], "anaya@example.com"]

const entries = Object.entries(student);
console.log(entries);  // [["name", "Ananya"], ["age", 22], ...]

entries.forEach(([key, value]) => {
    console.log(`${key}: ${value}`);
});

console.log("Total properties:", Object.keys(student).length);  // 5
