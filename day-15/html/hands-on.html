Day 15 — this, bind, call & apply

Hands-on

Open DevTools → Console. All tasks are in ../js/hands-on.js.

1 Predict the this

Type the following snippet exactly. 

const user = {

  name: "Priya",

  greet() {
    console.log(this.name);
  }
};

user.greet();

const g = user.greet;

g();

Predict the output BEFORE running.

Run the file and compare the actual output. 

// user.greet() works because this refers
// to the user object.
//
// g() loses the original object reference,
// so this becomes undefined in strict mode.

2 Fix it Three Ways

Start with this class: 

class Timer {

  constructor() {
    this.sec = 0;
  }

  tick() {
    this.sec++;
    console.log(this.sec);
  }
}

const t = new Timer();

setInterval(t.tick, 1000);

Observe the TypeError.

Now fix it using:

  • .bind()
  • Arrow wrapper
  • Class field arrow
// 1. bind()

setInterval(t.tick.bind(t), 1000);

// 2. Arrow wrapper

setInterval(() => t.tick(), 1000);

// 3. Class field arrow

class Timer2 {

  sec = 0;

  tick = () => {
    this.sec++;
    console.log(this.sec);
  };
}

const t2 = new Timer2();

setInterval(t2.tick, 1000);

Verify all three versions log: 1, 2, 3...

Reference: ../js/hands-on.js

3 call() / apply() / bind()

Define the following function: 

function describe(role, city) {

  console.log(
    `${this.name} is a ${role} from ${city}`
  );
}

const u = {
  name: "Aarav"
};

describe.call(u, "developer", "Hyderabad");

describe.apply(u, ["designer", "Mumbai"]);

const boundDescribe =
  describe.bind(u, "developer");

boundDescribe("Bangalore");

Observe how all three set the value of this. 

// call() passes arguments individually.
//
// apply() passes arguments as an array.
//
// bind() returns a new permanently bound function.

Reference: ../js/hands-on.js

4 Bonus — Arrow vs Regular Method

Create the following object: 

const team = {

  members: ["Priya", "Aarav", "Riya"],

  printRegular() {

    this.members.forEach(function (m) {
      console.log(this.members.length, m);
    });
  },

  printArrow() {

    this.members.forEach((m) => {
      console.log(this.members.length, m);
    });
  }
};

team.printRegular();

team.printArrow();

Observe which method breaks and why. 

// Regular functions get their own this,
// so printRegular() loses the team object.
//
// Arrow functions inherit this from the
// surrounding scope, so printArrow() works.
