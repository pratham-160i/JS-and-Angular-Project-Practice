JavaScript Learning

Learning journey from basics to advanced

Each day has three pages: Lesson (topics & demo code), Hands-on (in-class tasks), and Homework (take-home). Choose a day and page from the Curriculum outline on the right.

Expand a day, then use Page for the full lesson or lab view, or Code to open the matching .js file in a new tab so you can read or copy the source.

Tip: for the most reliable experience (all scripts and styles loading), serve this folder with a static server. Example: python -m http.server 9999 from this folder, then open http://127.0.0.1:9999/index.html in your browser (use another port if that one is busy).
