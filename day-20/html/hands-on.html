Task 3 Block the Event Loop
  • Create a busy-wait loop using Date.now().
  • Observe delayed timer execution.
  • Understand event loop blocking. 
function blockFor(ms) {

  const start = Date.now();

  while (Date.now() - start < ms) {

    // Busy wait
  }
}

setTimeout(() => {

  console.log("timer");

}, 100);

blockFor(2000);

console.log("after block");
// Output:
//
// after block
// timer
//
// The timer waits until the
// blocking synchronous code finishes.

Try the steps, then compare with ../js/hands-on.js.

Bonus Microtask Storm
  • Schedule multiple timers.
  • Add Promise microtasks inside one timer.
  • Observe how microtasks interrupt macrotasks. 
for (let i = 1; i <= 5; i++) {

  setTimeout(() => {

    console.log(`Timer ${i}`);

    if (i === 3) {

      Promise.resolve()

        .then(() => console.log("Microtask 1"))

        .then(() => console.log("Microtask 2"))

        .then(() => console.log("Microtask 3"));
    }

  }, 0);
}
// Example Output:
//
// Timer 1
// Timer 2
// Timer 3
// Microtask 1
// Microtask 2
// Microtask 3
// Timer 4
// Timer 5
//
// After every macrotask,
// the event loop fully clears
// the microtask queue first.

Try the steps, then compare with ../js/hands-on.js.
