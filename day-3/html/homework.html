Exercise 1: GST Calculator (Without Functions)

What to do:

  • Build a function-less GST calculator
  • Take const price = 1500 and const gstRate = 18
  • Compute and log price, GST amount, and total
const price = 1500;
const gstRate = 18;
const gstAmount = (price * gstRate) / 100;
const total = price + gstAmount;

console.log(`Price: ₹${price}`);
console.log(`GST (${gstRate}%): ₹${gstAmount}`);
console.log(`Total: ₹${total}`);

Exercise 2: Comparison Table (== vs ===)

What to do:

  • Compare 10 different value pairs with both == and ===
  • Make a table in a comment showing the differences
// Comparison Table: == vs ===
// Value Pair          | ==     | ===    | Reason
// ==================  ========================================
// 5 and "5"           | true   | false  | == coerces string to number
// 0 and false         | true   | false  | == coerces boolean to number
// null and undefined  | true   | false  | == special case, === strict
// "" and false        | true   | false  | == coerces both to 0
// "abc" and "abc"     | true   | true   | same value and type
// 1 and true          | true   | false  | == coerces true to 1
// [] and ""           | true   | false  | == coerces array to string
// 0 and "0"           | true   | false  | == coerces string to number
// NaN and NaN         | false  | false  | NaN is never equal to itself
// {} and {}           | false  | false  | different object instances

console.log("See comments above for detailed comparison table!");

Exercise 3: Adult or Minor Ternary

What to do:

  • Write a ternary that returns "adult" if age ≥ 18 else "minor"
  • Test with 5 different ages
const testAges = [10, 17, 18, 21, 65];

testAges.forEach(age => {
  const status = age >= 18 ? "adult" : "minor";
  console.log(`Age ${age}: ${status}`);
});

// Age 17: minor
// Age 18: adult
// Age 21: adult
// Age 65: adult

Exercise 4: Discount Calculator with && and Ternary

What to do:

  • Write a "discount calculator"
  • If user is a member AND total > 1000, give 10% discount
  • Else no discount. Use && and ternary
const isMember = true;
const total = 1500;

const discount = (isMember && total > 1000) ? (total * 10) / 100 : 0;
const finalAmount = total - discount;

console.log(`Total: ₹${total}`);
console.log(`Member: ${isMember}`);
console.log(`Discount: ₹${discount}`);
console.log(`Final Amount: ₹${finalAmount}`);

// Test case 2: Not a member
const isMember2 = false;
const total2 = 1500;

const discount2 = (isMember2 && total2 > 1000) ? (total2 * 10) / 100 : 0;
const finalAmount2 = total2 - discount2;

console.log(`\nTotal: ₹${total2}`);
console.log(`Member: ${isMember2}`);
console.log(`Discount: ₹${discount2}`);
console.log(`Final Amount: ₹${finalAmount2}`);
