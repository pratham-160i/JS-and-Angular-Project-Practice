Hands-on

Task 1 Pure or Impure?

Mark each function as pure or impure and explain why. 

// 1

const add =

    (a, b) =>

        a + b;

// PURE

// Same inputs always
// produce same output.

// No side effects.



// 2

const now =

    () => Date.now();

// IMPURE

// Depends on current time.
// Different outputs each call.



// 3

const sortArray =

    (arr) => arr.sort();

// IMPURE

// Mutates the original array.



// 4

const safeSort =

    (arr) =>

        [...arr].sort();

// PURE

// Creates a copy first.
// Original array unchanged.



// 5

const logValue =

    (x) => {

        console.log(x);

        return x;
    };

// IMPURE

// Causes side effect
// by writing to console.

Pure: 1, 4

Impure: 2, 3, 5

Task 2 pipe from Scratch

Implement pipe(...fns) and run functions left-to-right. 

function pipe(...fns) {

    return function(input) {

        return fns.reduce(

            (value, fn) =>

                fn(value),

            input
        );
    };
}

const addOne =

    (x) => x + 1;

const square =

    (x) => x * x;

const negate =

    (x) => -x;

const process =

    pipe(

        addOne,

        square,

        negate
    );

console.log(

    process(5)
);

Output:
-36

Task 3 Curry It

Create a curried volume function and partially apply values. 

const volume =

    (l) =>

        (w) =>

            (h) =>

                l * w * h;

const lengthOf5 =

    volume(5);

const lengthOf5width3 =

    lengthOf5(3);

console.log(

    lengthOf5width3(2)
);

Output:
30

Bonus Real Pipeline

Build a complete functional pipeline using only pure functions. 

const users = [

    {

        name: "priya",

        age: 25
    },

    {

        name: "aarav",

        age: 17
    },

    {

        name: "riya",

        age: 30
    }
];

function pipe(...fns) {

    return function(input) {

        return fns.reduce(

            (value, fn) =>

                fn(value),

            input
        );
    };
}

const adults =

    (users) =>

        users.filter(

            (u) =>

                u.age >= 18
        );

const capitalizeNames =

    (users) =>

        users.map(

            (u) => ({

                ...u,

                name:

                    u.name[0]
                    .toUpperCase()

                    +

                    u.name
                    .slice(1)
            })
        );

const sortByAgeDesc =

    (users) =>

        [...users].sort(

            (a, b) =>

                b.age - a.age
        );

const namesOnly =

    (users) =>

        users.map(

            (u) =>

                u.name
        );

const processUsers =

    pipe(

        adults,

        capitalizeNames,

        sortByAgeDesc,

        namesOnly
    );

console.log(

    processUsers(users)
);

Output:
["Riya", "Priya"]
